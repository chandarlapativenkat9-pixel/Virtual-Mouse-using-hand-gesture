AI Virtual Mouse

Control your computer with simple hand gestures using Python, OpenCV, and MediaPipe.

View on GitHub

How It Works

This project uses Artificial Intelligence to track hand landmarks in real-time. By calculating the distance between your Index finger and Thumb, the system determines whether to move the cursor or perform a click.

☝️ Cursor Control

Simply raise your Index Finger to move the mouse cursor across the screen. We use smoothing algorithms to ensure the movement is fluid and jitter-free.

👌 Click Detection

Pinch your Index Finger and Thumb together to perform a click. The system includes debouncing to prevent accidental double-clicks.

🚀 High Performance

Built with OpenCV and MediaPipe, the system runs efficiently on standard CPUs without needing a dedicated GPU.



Quick Start

To run this project on your local machine, run the following commands in your terminal:

git clone https://github.com/YOUR_USERNAME/virtual-mouse.git
pip install opencv-python mediapipe pyautogui numpy
python main.py
